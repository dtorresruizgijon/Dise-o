# Sitio web Nucleus e-wallet
- Realiza la maquetación de la landing page de un sitio web para pagos online.
## Requisitos 
- Escribe las hojas de estilo con Sass, utilizando todo lo que consideres conveniente para una codificación y organización más eficiente del código CSS, como:
  - separación de reglas en varios archivos (partials)
  - variables
  - anidación
  - mixins
  - interpolaciones
  - ...
- Utiliza CSS Grid para la maquetación de los elementos principales y Flexbox para alineaciones, direcciones de los elementos, etc.
- La página tiene un diseño responsive.
- Importante: claridad, estructuración del código y uso de comentarios.
- No está permitido el uso de IA durante el desarrollo del ejercicio (chatgpt, Gemini, Github Copilot, etc.).
- Vuestra pantalla será monitorizada y grabada durante la realización de la prueba.
  
## Guía de estilos
### Breakpoints
- 768px
- 992px
- 1280px
  
### Tipografía
- Tipo de letra principal: *Poppins*

- Dimensiones (si se define un font-size de 62.5% en el elemento html):
  - Textos
    - h1: 5rem
    - h2: 4.6rem
    - h3: 3rem
    - p: 1.6rem
  - Altura de línea: 1.8 para los párrafos y 1.2 para los titulares

### Colores
  - morado: #6834a6;
  - negro: #000000
  - blanco: #ffffff

### Contenedor
  - Anchura: 90%
  - Anchura máxima: 120rem

### Altura Header
  - 100px
  - 500px (anchura ventana navegador a partir de 768px)
  - 600px (anchura ventana navegador a partir de 992px) 
  - 1110px (anchura ventana navegador a partir de 1280px) 
  
## Rúbrica corrección
- Estructura carpetas y rutas
- Código (ordenado, indentado, comentado, nombres de clases representativos)
- HTML (estructurado y semántico)
- Aplicación de BEM
- Configuración y uso de fuente
- Uso imágenes
- Propiedades texto (tamaños, grosor, altura de línea)
- Centrado layout y uso containers
- Transiciones
- Dimensiones y espaciado (márgenes, paddings,...) 
- Uso de Flexbox (alineaciones, dimensionamiento, número de flex aplicados)
- Uso de Grid (definición cuadrícula, colocación, número de grids aplicados)
- Configuración media-querie
- Prueba funcional (responsive en todos los bloques)
- Variables Sass
- Anidación
- Modularización (partials)
- Mixins
- Aportaciones Sass
